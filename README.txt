JENNA LETTER WEBSITE — FINAL

The uploaded 10CM audio is already included as:
background-music.mp3

OPEN LOCALLY
1. Keep index.html and the assets folder together.
2. Open index.html in Chrome, Edge, Safari, or Firefox.
3. Tap “Open the letter.” The music begins through that tap and loops in the background.

PUBLISH ONLINE
Upload the entire jenna-light-letter-final folder to a static host such as Netlify, Cloudflare Pages, or GitHub Pages. Do not upload index.html alone.

BROWSER NOTE
Modern browsers block audible autoplay before a visitor interacts with a page. The opening button is intentionally used to trigger the audio reliably.
